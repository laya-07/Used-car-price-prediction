# Used Car Price Prediction

This project predicts the selling price of used cars using Machine Learning techniques. The project was developed using Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.

## Project Objectives
- Analyze used car dataset
- Perform data cleaning and preprocessing
- Apply feature engineering techniques
- Train a Linear Regression model
- Predict car selling prices accurately

## Features Implemented
- Exploratory Data Analysis (EDA)
- Missing value handling
- Feature engineering
- One-hot encoding
- Train-test splitting
- Feature scaling using StandardScaler
- Machine Learning pipeline creation
- Model evaluation using R² Score, MAE, MSE, RMSE
- Model saving using Joblib

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
